# 🚀 Nexus – Production-Ready Meeting Scheduler & Team Dashboard

Nexus is a modern, scalable, and production-ready meeting scheduler and team management dashboard built using **React 18**, **TypeScript**, and **Tailwind CSS**. It helps teams efficiently manage schedules, pitches, and user workflows through a clean and intuitive interface.

[![Vercel Deployment](https://img.shields.io/badge/Vercel-Deployed-000000?style=for-the-badge&logo=vercel)](https://nexus-yousaf.vercel.app)

---

## 🌟 Key Features
* **Secure Authentication:** Role-based Login/Signup system.
* **Dual-Role Dashboards:** Custom experiences for Entrepreneurs and Investors.
* **Interactive Scheduling:** FullCalendar integration for meeting management.
* **Pitch Creation Wizard:** Professional 3-step workflow for startup pitches.
* **Modern UI:** High-contrast glassmorphism design.

## 🛠 Tech Stack
* **Core:** React 18, TypeScript, Tailwind CSS, Vite.
* **Libraries:** FullCalendar, React Router v7, Lucide Icons.
